# inception
docker compose를 활용한 docker container 환경 구성 프로그램입니다.

## Key Features
- mariadb
- nginx
- wordpress + php8
<br>
with alpine linux

## How To Use
```bash
# Clone this repository
$ git clone https://github.com/todoni/inception

# Go into the repository
$ cd inception

# build
$ make

Password: 'your sudo password'
```
